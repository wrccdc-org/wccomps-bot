Role Synchronization

This will synchronize team roles from the volunteer guild to the competition guild.

  • Users with team roles in volunteer guild will get them in competition guild
  • Users without roles in volunteer guild will lose them in competition guild
  • This is a one-way sync from volunteer to competition

Note:

Role synchronization requires Discord bot access. Clicking the button below will notify the ops channel to run the sync command via the Discord bot.

Request Role Sync
Manual Sync via Discord

For immediate role synchronization, use the Discord bot command:

/admin sync-roles

This command must be run by an admin in the Discord server.
